1. Overview

Web Summary (Chrome Web Store name: A · Web Summary) is a browser extension that helps you decide whether to keep reading the active tab—then summarizes it using your own Google Gemini API key (BYOK — bring your own key).

There is no Web Summary backend server. The extension runs in your browser (side panel) and talks directly to Google’s Gemini API when you choose to summarize.

Single purpose: help you decide whether to read the current web page tab and summarize it for you.

2. Privacy Policy

What we do not collect or store

We do not collect, log, or store:

  • Page text or URLs you summarize
  • Generated summaries
  • Usage analytics tied to your browsing

What stays on your device

The only data the extension may persist locally is your Gemini API key (if you save it in Settings), your side panel UI language (English, Korean, or 中文), your Auto summarize preference (default OFF), your Auto summarize excluded domains list, and whether you have completed your first summary (onboarding only). All are stored in chrome.storage.local on this browser only.

What is sent to third parties

When you tap Summarize this tab, or when Auto is ON and you stay on a tab for about 3 seconds, article text is sent directly from your browser to Google’s Gemini API using your API key. We do not receive a copy of that content.

That processing is governed by Google’s Privacy Policy and Gemini API terms.

  • On unpaid (free) tier keys, Google may use API prompts and responses to improve its products (see “Unpaid Services” in the Gemini API terms).
  • If you are in the EEA, UK, or Switzerland, Google recommends a billing-enabled (Paid) project—see section 8.

Page access

A content script is injected only when a summary is requested (manual button or Auto after dwell) on the active http(s) tab, to extract readable text. It does not stay active on every page for background summarization.

Summary results

Results (read/skip verdict and three-line summary) appear in the side panel only and are not saved by the extension. The UI may show the source tab URL for reference; we do not persist URLs or summaries. Output may include translation when your UI language differs from the article. Closing the panel or browser clears them from the UI.

3. Terms of Use

  • Your API key: You are responsible for obtaining, securing, and managing your Gemini API key. Anyone with your key can use your Google AI quota. You must be 18 or older to use the Gemini API per Google’s terms.
  • Costs & quota: Summaries consume your Gemini API usage (free or paid tier per Google AI Studio). We do not bill you.
  • Translation in summaries: If your side-panel language differs from the article language, the model may produce a translated three-line summary. Use only on pages you are allowed to read and process.
  • AI output: Summaries are machine-generated. Verify important facts yourself; do not rely on them for legal, medical, or financial decisions. When sharing a summary, include the source URL and note that it is AI-generated.
  • Acceptable use: Use the extension in compliance with Google’s Gemini API terms, each website’s terms, and applicable law. Do not submit content you are not allowed to process (e.g., paywalled or restricted material you may not extract).
  • Auto summarize: When Auto is ON, the extension extracts text automatically after you stay on a tab (~3 seconds). Ensure this complies with sites you visit.
  • As-is: The extension is provided without warranties. We may update or discontinue it at any time.
Keep your API key private. Do not share screenshots or recordings while the key is visible in Settings.

6. Data & Limits

  • Only http(s) pages with enough readable text can be summarized.
  • Long pages: only the first portion of extracted text (about 8,000 characters) is sent to the model; the UI notes when content was truncated.
  • Requires Chrome 114+ (Side Panel API).

7. Copyright & Third-Party Content

Web Summary does not host, sell, or redistribute full articles. It shows a short AI-generated read/skip verdict and three lines for personal browsing decisions only.

  • Your responsibility: You must have the right to view the page and to send its text to Google’s API under that site’s terms and applicable copyright law.
  • Not a substitute for the original: Summaries may omit context or contain errors. Open the source tab for the full article.
  • Sharing: If you copy or share a summary, include the source URL and state that the text is AI-generated.
  • News & paid content: Many publishers restrict automated extraction or AI processing. Use manual summarize only where permitted; Auto may not be appropriate on some sites.

1. 概述

Web Summary（Chrome 网上应用店名称：A · Web Summary）是一款浏览器扩展，帮助您判断当前标签是否值得继续阅读，并使用您自己的 Google Gemini API 密钥（BYOK）进行摘要。

无 Web Summary 后端服务器。扩展在浏览器侧边栏运行，仅在您主动摘要时直接向 Google Gemini API 发送请求。

单一用途：帮助用户判断是否阅读当前网页标签，并为其提供摘要。

2. 隐私政策

我们不收集或保存的内容

我们不会收集、记录或保存：

  • 您摘要的页面正文或 URL

  • 与浏览行为绑定的使用分析数据

保留在本设备上的数据

扩展可能持久保存的数据：您在设置中保存的 Gemini API 密钥，侧边栏 界面语言（English、韩文或 中文），Auto 自动摘要开关（默认关闭），Auto 排除域名列表，以及是否 已完成首次摘要（仅用于引导）。均存储于本浏览器的 chrome.storage.local。

发送给第三方的数据

点击「摘要当前标签」，或开启 Auto 并在标签停留约 3 秒时，正文从您的浏览器直接发送至 Google Gemini API（使用您的密钥）。我们不会收到该内容的副本。

页面访问

仅在您请求摘要时（手动点击或 Auto 停留触发），向当前 http(s) 标签注入内容脚本以提取可读正文，不会在后台对所有页面常驻运行。

摘要结果

1. 개요

Web Summary(Chrome 웹스토어 이름: A · Web Summary)는 현재 탭을 계속 읽을지 판단하고, 사용자 본인의 Google Gemini API 키(BYOK)로 요약하는 Chrome 확장 프로그램입니다.

Web Summary 백엔드 서버는 없습니다. 확장은 브라우저 사이드패널에서 실행되며, 요약을 요청할 때만 Google Gemini API에 직접 연결합니다.

단일 목적: 현재 웹 페이지 탭을 읽을지 판단하고 요약을 제공하는 것.

2. 개인정보 처리방침

수집·저장하지 않는 항목

다음을 수집·기록·저장하지 않습니다:

  • 요약한 페이지 본문 또는 URL
  • 생성된 요약·읽기/건너뛰기 판정
  • 브라우징과 연결된 사용 분석

기기에 남는 데이터

확장이 로컬에 저장할 수 있는 항목: 설정에 저장한 Gemini API 키, 사이드패널 UI 언어(English·한국어·中文), Auto 요약 토글(기본 OFF), Auto 제외 도메인 목록, 첫 요약 완료 여부(온보딩용). 모두 이 브라우저의 chrome.storage.local에만 저장됩니다.

제3자로 전송되는 데이터

「현재 탭 요약하기」를 누르거나 Auto가 켜져 탭에 약 3초 머물면, 본문이 브라우저에서 사용자 API 키로 Google Gemini API에 직접 전송됩니다. Web Summary는 해당 내용의 사본을 받지 않습니다.

처리에는 Google 개인정보처리방침과 Gemini API 약관이 적용됩니다.

  • 무료(Unpaid) tier 키는 Google이 API 입출력을 제품 개선에 활용할 수 있습니다(Gemini API 약관 Unpaid Services 참고).
  • EEA·영국·스위스 거주 시 billing 연결(Paid) 프로젝트를 권장합니다—8절 참고.

페이지 접근

콘텐츠 스크립트는 요약 요청 시에만(수동 또는 Auto 체류 후) 활성 http(s) 탭에 주입되어 읽을 수 있는 본문을 추출합니다. 모든 페이지 로드마다 백그라운드로 실행되지 않습니다.

요약 결과

결과(읽기/건너뛰기·세 줄 요약)는 사이드패널에만 표시되며 확장이 저장하지 않습니다. UI에 출처 탭 URL이 표시될 수 있으나 URL·요약을 영구 저장하지는 않습니다. UI 언어와 기사 언어가 다르면 번역이 포함될 수 있습니다. 패널이나 브라우저를 닫으면 UI에서 사라집니다.

3. 이용약관

  • 개인 triage 목적: 이미 연 탭을 읽을지 판단하는 도구입니다. 전체 번역 서비스, 뉴스 재배포·집계 도구가 아닙니다(언어가 다를 때 짧은 번역 행이 나올 수 있음).
  • API 키: Gemini API 키의 발급·보관·관리는 사용자 책임입니다. 키를 가진 사람은 Google AI 할당량을 사용할 수 있습니다. Google 약관상 만 18세 이상이어야 합니다.
  • 비용·할당량: 요약은 Google AI Studio의 무료/유료 할당량을 사용합니다. Web Summary는 요금을 청구하지 않습니다.
  • 요약 내 번역: 사이드패널 언어와 기사 언어가 다르면 모델이 번역된 세 줄 요약을 만들 수 있습니다. 읽기·처리 권한이 있는 페이지에서만 사용하세요.
  • AI 출력: 요약은 기계 생성입니다. 중요한 사실은 직접 확인하세요. 법률·의료· 재무 결정에 의존하지 마세요. 공유 시 출처 URL과 AI 생성임을 밝히세요.
  • 허용 사용: Gemini API 약관, 각 웹사이트 이용약관, 관련 법률을 준수하세요. 처리 권한이 없는 콘텐츠(무단 추출이 금지된 유료 콘텐츠 등)를 제출하지 마세요.
  • Auto 요약: Auto ON 시 탭에 약 3초 머물면 자동으로 본문을 추출합니다. 방문 사이트 규정을 확인하세요.
  • 현 상태 제공: 보증 없이 제공되며, 언제든 업데이트·중단할 수 있습니다.
API 키를 비공개로 유지하세요. 설정에 키가 보이는 상태로 스크린샷·녹화를 공유하지 마세요.

6. 데이터 및 제한

  • 충분한 본문이 있는 http(s) 페이지만 요약 가능.
  • 긴 페이지: 추출 본문 앞부분(약 8,000자)만 전송; UI에 잘림 표시.
  • Chrome 114+ 필요(Side Panel API).

7. 저작권 및 제3자 콘텐츠

Web Summary는 기사 전문을 호스팅·판매·재배포하지 않습니다. 개인적 읽기 판단을 위한 짧은 AI 판정과 세 줄만 표시합니다.

  • 사용자 책임: 해당 페이지를 볼 권한이 있고, 사이트 약관·저작권법에 따라 Google API로 본문을 보낼 수 있어야 합니다.
  • 원문 대체 아님: 요약은 맥락을 빠뜨리거나 오류가 있을 수 있습니다. 전문은 원문 탭에서 확인하세요.
  • 공유: 요약을 복사·공유할 때 출처 URL과 AI 생성임을 함께 표기하세요.
  • 뉴스·유료 콘텐츠: 많은 언론사가 자동 추출·AI 처리를 제한합니다. 허용된 경우에만 수동 요약을 사용하고, 일부 사이트에서는 Auto가 부적절할 수 있습니다.

8. Gemini API (BYOK)

본 확장은 BYOK(자체 키) 클라이언트입니다. API 호출은 사용자 키로 사용자 브라우저에서 이루어집니다. Web Summary는 요청을 대리하지 않으며 페이지 본문을 서버에 저장하지 않습니다.

사용자 의무

  • AI Studio에서 키를 만들면 사용자가 Gemini API 추가 약관과 Google API 서비스 약관에 동의한 것입니다.
  • 생성형 AI 금지 사용 정책을 포함해 Google 정책을 준수해야 합니다.
  • Google 약관은 Google AI 모델을 전문·상업 목적으로 사용하는 개발자를 대상으로 합니다. BYOK에서는 사용자가 직접 키를 만들고 해당 약관 하에 사용합니다.

EEA·영국·스위스

Google 약관에 따르면 EEA·영국·스위스 사용자에게 API 클라이언트를 제공할 때 Paid Services(Cloud Billing이 연결된 프로젝트)를 사용해야 합니다. 무료 할당량을 쓸 수 있으나 데이터 처리는 Paid Service 규칙을 따릅니다. Gemini API billing을 참고해 해당 지역이면 프로젝트에 billing을 활성화하세요.

무료(Unpaid) tier

키가 Google 무료(미결제) 할당량을 쓰면, 제출한 프롬프트·응답이 제품 개선에 사용될 수 있고 인간 검수자가 처리할 수 있습니다( Gemini API 약관 Unpaid Services 참고). 민감·기밀 정보는 무료 tier에 제출하지 마세요. billing 연결(Paid tier) 시 Google의 API 데이터 사용 방식이 달라집니다.

Web Summary는 Google과 무관합니다. Google은 API 약관·할당량·집행을 언제든 변경할 수 있습니다.
